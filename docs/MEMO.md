
* 本プロジェクトはGodotで作成されるゲームである
* 基本はスーパーマリオのようなアクションゲームとテラリアのようなサイドビューのサンドボックスゲームを合体したようなゲームである。
* ドット絵（ピクセルアート）のレトロな雰囲気である
* 墜落した宇宙船がスタートポイントで、ここから上下左右にワールドが広がっている

# ワールド
* ワールドはシード値によって生成される
* ワールドのサイズは有限。ワールドをまたがって旅をすることを前提とする。

# バイオーム
* ワールドは複数のバイオームで構成されており、バイオーム同士は厳密な境界線はなく緩やかに遷移する
* そのため、バイオーム同士で隣り合わせになれないものもある
* 初期バイオームは平地
* バイオームが多いこともゲームの特徴にしたい

# アクション
* 主人公の宇宙服にクラフトによりいろいろなオプションを足して強くなっていく
* ジャンプ、ダッシュ、攻撃などの基本的なアクション
* 敵は踏むと倒せる（ことが多い。マリオを参考に）

# モンスター
* バイオームごとにモンスターがきまっている
* 中心の宇宙船からの距離に応じてモンスターが強くなっていく
* マリオなどの明るいゲームを参考に、あまりリアル寄りではなくデフォルメされたモンスターを作る
* SFなのでロボット型や、エイリアン型などもいてよい
* 中ボスなどもそのうち実装したいが初期リリースでは未定

# 部屋
* 部屋の概念があり、プレイヤーが閉空間を作成するとそこは部屋とみなされる。
* 部屋に配置された調度品によって部屋の種類とレベルが決まる
* 体力は寝室にいると自然回復する
* 食料は食堂で調理して食べることができる
* 扉をつかうと兵空間の中と外を出入りできる
* モンスターは基本的には部屋には入ってこない（例外はあり）
* バイオームに特徴的な部屋が廃棄された（壊れた）状態で生成されることもある

## 部屋案
* 寝室（ベッドとチェスト）
* 作業場（作業台とチェスト）
* 食堂（キッチンとテーブルと椅子）
* 教会（祭壇とベンチ）
* 図書館（本棚と机と椅子）
* 武器庫（武器ラックとチェスト）
など


# 宇宙船
* ワールドの中心に墜落しており、寝室と作業台からなる基本的な部屋を持つ
* 破壊されない特別な建物扱い
* 操縦室という特別な部屋がある。ここはほかのワールドへのワープポイントになる。最初に入ったワールドをメインワールドとみなし、サブワールドとして、シード値をいくつか保持でき、ワールドを行き来できるものとする。
